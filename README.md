# Word 格式化工具 MVP

一个简单易用的 Word 文档自动排版工具，支持用户自定义格式规则。

## 功能特点

- 🎯 **智能标题识别**：自动识别文档中的标题段落，用户可确认调整
- ⚙️ **自定义格式规则**：支持自定义标题和正文的字体、字号、对齐方式等
- 📥 **一键下载**：排版完成后直接下载新文档

## 安装步骤

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
streamlit run app.py
```

### 3. 打开浏览器

程序会自动打开浏览器访问 `http://localhost:8501`

## 使用方法

1. **上传文档**：选择需要排版的 .docx 文件
2. **设置格式**：自定义各级标题和正文的格式
3. **确认标题**：程序会识别可能是标题的段落，用户可以确认或调整
4. **下载结果**：点击下载按钮获取排版后的文档

## 标题识别规则

程序会根据以下规则识别标题：

1. **格式匹配**：
   - "第X章/节" → 一级标题
   - "一、二、三、" → 一级标题
   - "（一）（二）" → 二级标题
   - "1. 2. 3." → 三级标题
   - "1.1 1.2" → 三级标题

2. **格式特征**：
   - 字号较大（≥14pt）→ 标题
   - 加粗文本（长度<50字）→ 标题

用户可以在界面上调整识别结果。

## 注意事项

- 仅支持 .docx 格式
- 复杂表格和图片可能需要手动调整
- 建议先用简单文档测试
